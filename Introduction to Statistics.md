# 1강

- Random(형용사스러움) = Probablity = Likelihood
- 표본공간 S = outcome들의 집합
  - {(1. 1), .., (6, 6)}
  - {HH, HT, TH, TT}
- 조건부확률
  - 정의 : 표본공간 S를 특정한 event들의 모임으로 제한
  - Bayes' Rule
- 확률변수 is 함수 s.t 표본 공간에서 정의된 실변수 함수 X : S -> R
  - X(HH) = 2
  - P(X = 2) = 1 / 4
  - 확률변수의 정의역에 따라
    - `discrete` if countable -> pmf = P(X = x) = 1/ 4
    - `conti` if uncountable -> pdf = P(a < X < b) = 1 / 4
  - cdf(cummulative)
